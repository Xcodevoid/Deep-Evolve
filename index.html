Quick Start

Get Started in 3 Easy Steps

$ git clone https://github.com/xcodevoid/Deep-Evolver

$ cd Deep-Evolver
$ pip install -r requirements.txt
$ # Edit config.json to enable/disable features
$ python run.py --config config.json

Configuration Options:

  • memory_enabled: Toggle persistent memory system
  • emotion_enabled: Enable emotional intelligence features
  • reflection_enabled: Activate self-reflection loop
  • verification_enabled: Turn on knowledge verification
  • creativity_enabled: Enable creativity dial controls
  • debugging_enabled: Activate interactive debugging
